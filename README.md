# Archaeological assistant
## Install
1. [install cargo](https://doc.rust-lang.org/cargo/getting-started/installation.html)
2. install crate:
    
    ```
   cargo install --git 
   ```
3. run:
    
    - mac

      ```
      open -a "~/.cargo/bin/Archaeological_Assistant.app"
      ```
    - linux or windows
    
        ```
      ~/.cargo/bin/Archaeological_Assistant.app
      ```
